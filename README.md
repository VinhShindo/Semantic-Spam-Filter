# Semantic-Spam-Filter

Semantic-Spam-Filter is a Python-based project for building and evaluating a **semantic text spam classifier** using NLP and machine learning techniques. The system includes data processing, model training, comparison between vectorizers and classifiers, and a simple web interface for inference.
